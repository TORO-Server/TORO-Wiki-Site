TORO鯖に本社・支社を置く企業体（株式会社・特殊法人等）の一覧です。  
  
自治体については[自治体一覧](../04_World_and_Cities/%E8%87%AA%E6%B2%BB%E4%BD%93%E4%B8%80%E8%A6%A7.md)を、鉄道会社については[鉄道事業者一覧](../03_Transportation/%E9%89%84%E9%81%93%E4%BA%8B%E6%A5%AD%E8%80%85%E4%B8%80%E8%A6%A7.md)、航空会社については[航空事業者一覧](../03_Transportation/%E8%88%AA%E7%A9%BA%E4%BA%8B%E6%A5%AD%E8%80%85%E4%B8%80%E8%A6%A7.md)をご覧ください。   
  
なお、企業体については、**この鯖において一切の権限を有しません**。
