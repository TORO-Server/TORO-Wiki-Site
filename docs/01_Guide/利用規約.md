### 利用規約（およびサーバールール）

#### 定義

* 本サービスを利用した時点ですべての利用規約に同意したものとみなします。
* 以下において、Minecraftサーバー「TORO SERVER」、「TORO SERVER 公式Discord」および「TORO SERVER 公式Wiki」を統括したサービスであるTORO SERVERを本サービス、Minecraftサーバー「TORO SERVER」をサーバー、「TORO SERVER 公式Discord」を公式Discord、「TORO SERVER 公式Wiki」を公式Wikiと呼称します。

##### 役職

* TORO SERVERにおける役職は上位からOwner、Administrator、Moderator、Supporterとします。
  + Ownerはサーバー所有者であるtoroponのみが取得します。
  + Administratorは本サービスの理念を理解し、Ownerの権限の一部が委託されています。Adminと略称します。
  + Moderatorには本サービスを安定的な稼働するための一部の権限が認められています。Modと略称します。
  + Supporterは運営チームを補佐する役割をもち、簡単な対応や情報収集、運営への報告を行います。Supと略称します。
* 役職の中で、特にOwner、Admin、Modは運営メンバーとみなされます。運営メンバーは運営会議に参加することができます。

##### 保護

* サーバー内でプレイヤーによって制作・建築・造成等することを制作と呼称し、そのようにして制作されたものを制作物と定義します。
* プラグイン「WorldGuard」によって建築を制限することを「保護する」といい、建築制限や建築制限区域を「保護」と定義します。
* 保護には保護の所有者（Owners）と保護のメンバー（Members）が設定されています。保護の所有者は、所有している保護内のすべての制作物についての権限を持っています。（他人の制作物であっても、保護の所有者の制作物と同様に扱われ、撤去・移設・改造等を自由に行えます。）

#### サーバールール

* 本サービスにおいて以下の行為をした場合、BAN（本サービスの利用停止）などの措置が取られる場合があります。
* また、悪質な場合は提携サーバー(Milkyway MC)との共有BANも行われ、提携サーバーでのサービス利用停止も行われる場合があります。
  + 他人の制作物（町を含む）を無断で破壊・改変する行為
  + 他人の制作物を無断で保護する行為
  + 他人の制作物を無断で複製する行為
  + 性的・差別的な制作
  + 無意味に大規模な保護をする行為
  + チャットで連続して大量の発言をする（チャットスパム）行為
  + 日本の法規に違反する行為
  + 性的・差別的または誹謗中傷と受け取られかねない発言・暴言
  + 他人に対して脅迫、圧迫、あるいは圧力をかける行為
  + 特定のプレイヤーに嫌がらせをしたり、正当な理由なく排除したりする行為
  + 荒らしに対して通報以外の反応をする行為
  + 危険なURLをチャットに貼る行為
  + 他人の個人情報を過度に要求する行為
  + 頼まれていないのに教える行為（教え魔行為）
  + チートMOD等を悪用する行為
  + MOBを大量に出し放置する行為
  + 執拗に要求する行為
  + 当サーバーの名誉・信頼を失墜させる行為
  + 他人に付きまとう（ストーカー）行為
  + 他人を監禁する行為
  + 不当に入手したアカウントでのログイン
  + 本サービス内における他マインクラフトサーバーの宣伝・勧誘行為
  + サーバーに過度の負担をかける行為
  + その他運営が悪質と判断した行為
* 本サービスを利用する者が本サービス外において以下の行為をした場合、BAN（本サービスの利用停止）などの措置が取られる場合があります。
  + 日本の法規に違反する行為
  + 性的・差別的または誹謗中傷と受け取られかねない発言・暴言
  + 他人に対して脅迫、圧迫、あるいは圧力をかける行為
  + 特定のプレイヤーに嫌がらせをしたり、正当な理由なく排除したりする行為
  + 他人の個人情報を過度に要求する行為
  + 執拗に要求する行為
  + 頼まれていないのに教える行為（教え魔行為）
  + 当サーバーの名誉・信頼を失墜させる行為
  + 他人に付きまとう（ストーカー）行為
  + 勧誘が禁止されている他サーバーにおけるTORO Serverへの勧誘行為
  + Japan Minecraft Serversへの複数のアカウントを利用した多重投票
  + その他運営が悪質と判断した行為
* 罰則はすべて各アカウントに対して適用されます（プレイヤーが誰かを問いません）。

#### 免責事項

* 本サービスを利用したことによって生じたいかなる損害についても、サーバー所有者はその責任を負いません。
* サーバー所有者は本サービス終了についての補償をする義務を持ちません。

#### 管理者権限

* サーバー内に存在するすべての建築物について、サーバー所有者はこれを無断で移設・改造または撤去することができます。
* ルールの詳細な適用範囲については、すべてサーバー所有者の裁量にのみ依るものとします。

#### 利用規約の変更

* 利用規約は変更されることがあります。変更については1週間以上前に本サイトで予告します。

### 推奨されないこと

サーバールールではありませんが、他プレイヤーと良い関係を保つために以下の行為はなるべく控えるか、十分に考えてから行いましょう。

* 先に建築している人のすぐ近くに全く異なる雰囲気の建築をすること
* アカウントの貸し借り
* 大規模な整地・埋め立て

#### 鉄道関係

* 何もない場所に向かって鉄道を敷設する行為
* 過度に線路を並行して建設する行為
* 許可なく他人の建築の近くに線路を敷設する行為
