**Tips**
> コマンド入力の途中で Tabキー を押すと補完されます（複数ある場合は再度押すことで変える事ができます）。  
> コマンド以外では接続しているプレイヤーの名前が補完されます。

  
  

### システム系

| コマンド | アクション | 備考 |
| --- | --- | --- |
| /help [1] | コマンドのヘルプを表示 | [1]:知りたいコマンド（空白にすると一覧が表示される） 短縮形:/? |
| /gamemode [1] | ゲームモードを変更 | [1]:(0/1/2/3/s/c/a/sp/survival/creative/adventure/spectator) |
| /god | ダメージを受けなくする | OFFにする時はもう一回 /god |
| /fly | クリエイティブ飛行できるようにする | OFFにする時はもう一回 /fly |
| /nttoggle | 自分のネームタグを非表示にする | 表示にする時はもう一回 /nttoggle |
| /wesv toggle | WorldEditの選択範囲を非表示にする | 表示にする時はもう一回 /wesv toggle |
| /heal | 体力回復 |  |
| /speed [1] | 移動速度を変更 | [1]:数値（0〜10） デフォルトは1 クリエイティブ飛行中に打つと飛行スピードを変更できる |
| /weather [1] | 天候を変更 | [1]:天気の名前（英語） |
| /time set [1] | 時刻を変更 | [1]:数値または時間の名前（英語） |
| /mail send [1] [2] | メールを送る | [1]:相手のID [2]:内容 |
| /mail read | メールを読む |  |
| /effectgui | ポーションの効果を自分に付与する | 表示されたGUIから選択 短縮形:/em |
| /workbench | その場で作業台を開く | 短縮形:/wbench, /craft |
| /enderchest | その場でエンダーチェストを開く | 短縮形:/ec |
| /afk | 退席状態にする | 移動したりチャットを打ったりすると解除される |
| /list | 接続しているプレイヤーの一覧を表示 | 退席状態も表示される |
| /seen [1] | 最後にログアウトした時間を表示 | [1]:プレイヤー名 |
| /history [1] | プレイヤー名の変更履歴を表示 | [1]:プレイヤー名 |
| /music | マイクラ内で音楽を聴く（マイクラ音源） | 表示されたGUIから選択 |
| /tps | サーバーのTPSを表示 | 最高:20 17を下回ると危険 |
| /ping | ポン！ | /pong |
| # | 文章の最初につけるとローマ字変換がされなくなる |  |

### WorldEdit

*WorldEditは権限が無いと使えません。使用したい場合は運営にお知らせください。*

| コマンド | アクション | 備考 |
| --- | --- | --- |
| //set [1] | 指定した範囲にブロックを設置 | [1]:ブロック名またはブロックID |
| //cut | 指定した範囲を切り取り |  |
| //copy | 指定した範囲をコピー |  |
| //paste | 切り取り・コピーしたものを貼り付け |  |
| //stack [1] | 指定した回数だけ伸ばす | [1]:回数(1) |
| //hcyl [1] [2] | **中空の**円を生成する | [1]:ブロック [2]:半径 |
| //cyl [1] [2] | 円を生成する | [1]:ブロック [2]:半径 |
| //hsphere [1] [2] | **中空の**球体を生成する | [1]:ブロック [2]:半径 |
| //sphere [1] [2] | 球体を生成する | [1]:ブロック [2]:半径 |
| //rotate [1] | クリップボードを回転させる | [1]:角度(時計回り) |
| //flip | クリップボードを反転する |  |
| //replace [1] [2] | [1]を[2]に置き換える | [1][2]:ブロック |
| //schematic save [1] | schemで選択範囲をセーブする | [1]:名前 |
| //schematic load [1] | schemをクリップボードに読み込む | [1]:名前 |
| //schematic list [1] | schemのリストを表示する | [1]:ページ(1) |
| //shift [1] | 現在の選択範囲をずらす | [1]:距離 |

| タグ | イベント |
| --- | --- |
| -a | 空気を対象にしない |
| -s | pasteやstackした所を選択 |
| -m [ブロック] | 指定したブロックを対象にする |

  
詳しくはこちら→[WorldEdit](../02_Plugins/WorldEdit.md)
