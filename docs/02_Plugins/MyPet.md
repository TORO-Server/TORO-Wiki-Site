### 使い方

#### ペットを飼う方法

手綱(リード)を持って飼いたいMOBを殺すと飼うことができます。  
また一部MOB(ボス等)は飼うことができません。

#### ペットを手放す方法

/petreleaseを実行した後、チャット欄でペットの名前をクリックすると手放せます。

### コマンド一覧

| コマンド | 説明 |
| --- | --- |
| /petinfo | ペットの情報を表示します |
| /pi | ペットのインベントリを開きます(サバイバルモード時のみ開けます) |
| /pcst <スキル名> | ペットのスキルツリー(技能)を設定します。 |
| /petname <付けたい名前> | ペットの名前を設定します |
| /psa | ペットを収納します |
| /pc | ペットを呼び出します |
| /petbehavior <行動名> | ペットの行動パターンを変更する(スキルによってできるものとできないものがある) |

### スキルツリー(技能)一覧

| スキル名 | 説明 |
| --- | --- |
| Combat | 戦闘に特化したスキルです |
| Utility | インベントリやアイテム収集などができるスキルです |
| PvP | PvPに特化したスキルです |
| Ride | 騎乗に特化したスキルです(遠距離攻撃しかしなくなる) |
| Farm | 戦闘に特化したスキルです(遠くの敵に弓を使うようになる) |

### 行動パターン一覧

スキルによって できないものもある  

| スキル名 | 説明 |
| --- | --- |
| Friendly | 待機モード(プレイヤーが攻撃した敵を攻撃するオオカミみたいな行動をする) |
| Normal | ノーマルモード(待機と同じ気がする) |
| Duel | 対決モード(待機と同じ気がする) |
| Raid | 攻撃モード(待機と同じ気がする) |
| Aggressive | 無差別攻撃モード (持ち主以外 見境なく攻撃する) |
| Farm | モンスター警戒態勢モード(モンスターだけ攻撃する) |
