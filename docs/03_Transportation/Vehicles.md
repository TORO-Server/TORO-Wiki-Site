乗り物を追加するプラグイン  
  
※乗り物を使用するには簡単な教習を受け、driver権限を得る必要があります。  

### コマンド

| コマンド | 種類 \<type\> | 説明 |
| --- | --- | --- |
| /getBike \<type\> | BLACK ORANGE RED | バイクを入手 |
| /getCar \<type\> | BLACK YELLOW MAGENTA | 自動車を入手 |
| /getTrain \<type\> | T1 T2 T3 | 汽車を入手 |
| /getRaft \<type\> | OLD NETHER PIRATE | 筏を入手 |
| /getPlane \<type\> | P1 P2 P3 | 飛行機を入手 |
| /getParachute \<type\> | PC1 PC2 PC3 | パラシュートを入手 |
| /getTank \<type\> | T1 T2 T3 | 戦車を入手 |
| /getHelicopter \<type\> | H1 H2 H3 | ヘリコプターを入手 |
| /getSubmarine \<type\> | S1 S2 S3 | 潜水艦を入手 |
| /getHoverBike \<type\> | HB1 HB2 HB3 | ホバーバイクを入手 |
| /getDrill \<type\> | D1 D2 D3 | ドリルを入手 |
| /getRacingCar \<type\> | YELLOW RED PURPLE | レーシングカーを入手 |
| /getSportBike \<type\> | SB1 SB2 SB3 | スポーツバイクを入手 |
| /getTractor \<type\> | YELLOW RED GREEN | トラクターを入手 |
| /getMecha \<type\> | MKI MKII MKV | すごい機械を入手 |

### 乗り物の特徴

| 種類 | 特徴 |
| --- | --- |
| バイク | ・ハーフブロックや階段を自動で登れる ・ジャンプができる ・二人乗りできる |
| 自動車 | ・ハーフブロックや階段を自動で登れる ・バックができる ・二人乗りできる |
| 飛行機 | ・離陸可能な速度のとき上を向くと空を飛べる ・TNTを持って右クリックするとTNTを投下できる ・二人乗りできる |
| 戦車 | ・ファイヤーチャージを持って右クリックすると前方に発射できる |
| ヘリコプター | ・ジャンプキーを押すと上に上昇する ・ファイヤーチャージを持って右クリックすると前方に発射できる |
| 潜水艦 | ・水の中でしか置けない ・乗った状態で右クリックするとフロントカメラに切り替わりジャンプキーを押すと解除できる |
| ホバーバイク | ・水に沈まない(水上を移動できる) ・1mのブロックを飛び越えることができる |
| ドリル | ・ブロックを掘れる |
| すごい機械 | ・ジャンプができる ・フックとレーザーを発射できる |
