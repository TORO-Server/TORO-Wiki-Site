### 概要

TORO Serverは自由な創作の場を追求するマインクラフト・クリエイティブサーバーです。  
ワールド内では自由に土地を確保し、WorldEdit等プラグインを利用した自由で快適な創作活動ができます。  
  
まずは**[ルール](../01_Guide/%E3%83%AB%E3%83%BC%E3%83%AB.md)**をお読みください。

### サーバー運営理念

#### 自由

基本的に何を作っても良いし、どんな遊び方をしても良い。いつ来ても良いし、いつ出ていっても良い。ただし他人の自由を制限・侵害する行為は原則制限される。

#### だれでも歓迎

初めてサーバーを訪れようとする人にとっての障壁を可能な限りなくすよう努める。

#### 標準的クリエイティブサーバー

TORO SERVERは通常のクリエイティブワールドの上位互換的存在であり、通常のクリエイティブワールドでできることはすべてできるようにする。
